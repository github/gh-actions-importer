# CypressIo/Cypress Write Workspace

```yaml
orbs:
  cypress: cypress-io/cypress@x.y
jobs:
  cypress-example:
    docker:
      - image: 'cimg/base:stable'
    steps:
      - cypress/write_workspace
```
