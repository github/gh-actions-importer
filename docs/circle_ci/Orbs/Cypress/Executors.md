# CypressIo/Cypress Executors

## CircleCI input

```yaml
orbs:
  cypress: cypress-io/cypress@x.y
jobs:
  cypress-example:
    executor: cypress/base-6
    steps:
      - cypress/run
```

### Transformed Github Action

```yaml
jobs:
  cypress-example:
    runs-on: ubuntu-latest
    container:
      image: cypress/base:6
    steps:
    - uses: actions/checkout@v2
```
