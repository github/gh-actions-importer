<disabled>false</disabled>

<blockBuildWhenUpstreamBuilding>false</blockBuildWhenUpstreamBuilding>

### Transformed Github Action

```yaml
    uses: actions/setup-node@v2
    - name: Run Node Command
      shell: node {0}
      run: |
        function hello(){
            console.log("Hello")
        }
        hello()
```

### Unsupported Options

- None

## Jenkinsfile pipeline
