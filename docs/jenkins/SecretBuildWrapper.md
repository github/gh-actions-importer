### Transformed Github Action

```yaml
env:
  TEST_TEXT: "${TEST_PARAM}"
```

### Unsupported Options

- Certificate
- SSH User private Key
- Zecret ZIP File
- Secret File
- Username and password (conjoined)
- Username and password (separated)
